A generic package for computing both forward and backward kinematics for arms. Developed as an alternative to pr2_arm_kinematics for people not using the PR2. This package is a fork of urdf_tools/arm_kinematics. It adds mechanisms to configure some of the KDL solver parameters and thus allows for greater flexibility.
A generic package for computing both forward and backward kinematics for arms. Developed as an alternative to pr2_arm_kinematics for people not using the PR2. This package is a fork of urdf_tools/arm_kinematics. It adds mechanisms to configure some of the KDL solver parameters and thus allows for greater flexibility.
